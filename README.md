# アンチヒーロー

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクター魔王が城を攻めてきた勇者一行を迎え撃つゲーム。

##ゲームの実装
###共通基本機能
* 背景画像と主要キャラの実装

### 担当追加機能
* スコアとレベルアップ機能:スコアの表示とスコアに応じてレベルが上がり魔王が進化する機能
